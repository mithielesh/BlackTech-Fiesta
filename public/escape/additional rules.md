**Level 1 – Tech Riddle (3 Minutes)**



Objective:

Solve a short technical riddle to unlock the next level.



**Participant:**



Read the riddle carefully.



Submit the exact correct answer before the timer ends.



No tab switching allowed.



**Admin:**



Predefine the exact correct answer.



Monitor timer and rule violations.



Unlock Level 2 only for correct submissions.



**Evaluation:**



Correct answer → Unlock Level 2.



Timeout → Eliminated.



No scoring. Pure survival gate.



**Level 2 – Visual Drag \& Drop (4 Minutes)**



Objective:

Arrange given technical items in correct logical order.



**Participant:**



Analyze the shuffled items.



Type the full item names in correct comma-separated order.



Submit before time expires.



**Admin:**



Predefine exact correct sequence.



Validate exact text match.



Monitor timer and tab switching.



**Evaluation:**



Exact correct order → Unlock Level 3.



Incorrect answer at timeout → Eliminated.



No partial credit. Survival gate.



**Level 3 – Logic Lock / Code Crack (5 Minutes)**



Objective:

Solve multiple technical clues and generate a final code.



**Participant:**



Solve all clues.



Apply the given final code rule.



Enter the exact final code.



Limited attempts allowed (if defined).



**Admin:**



Store correct final code in backend.



Validate exact match.



Monitor rule violations.



**Evaluation:**



Correct code → Unlock Level 4.



Timeout or failure → Eliminated.



No cumulative scoring. Survival gate.



**Level 4 – Multi-Constraint Tech Crisis Final (6 Minutes)**



Objective:

Analyze a real-world technical crisis and take correct actions.



**Participant:**



Read the crisis scenario carefully.



Select the required number of actions.



Arrange them in correct execution order.



Submit within time.



**Admin:**



Predefine correct action set and exact order.



Lock submission after timer expires.



Monitor rule violations.



**Evaluation:**



Only fully correct sequence qualifies.



If multiple teams solve correctly:

→ Fastest correct teams proceed to Level 5.



Incorrect or timeout → Eliminated.



This is the competitive survival round.



**Level 5 – HTML/CSS/JS Coding Final (10 Minutes)**



Objective:

Implement vertical-to-horizontal scroll synchronization.



**Participant:**



Use provided webpage template.



Write required CSS and JavaScript.



Do not modify base HTML structure.



No external libraries allowed.



Submit working solution within time.



**Admin:**



Provide starter template.



Test functionality live.



Validate correct proportional scroll behavior.



**Evaluation:**



Only fully working implementation qualifies.



Among correct teams, fastest submission wins.



This determines final winner.
